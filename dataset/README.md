# Датасет Яндекс.Директ

## Источник

Реальный выгрузка рекламных кампаний из Яндекс.Директа.

## Размер

- **Строк:** 33 606
- **Колонок:** 16

## Структура колонок

| Колонка | Тип | Описание |
|---|---|---|
| `День` | Дата (дд.мм.гггг) | Дата показа |
| `№ Кампании` | UInt64 | ID кампании |
| `Название кампании` | String | Название рекламной кампании |
| `Название группы` | String | Название группы объявлений |
| `№ Группы` | UInt64 | ID группы |
| `№ Объявления` | UInt64 | ID объявления |
| `Регион местонахождения` | String | Регион показа |
| `Регион местонахождения.1` | UInt32 | Код региона |
| `Ключевая фраза` | String | Ключевое слово |
| `Расход, ₽` | Float64 | Затраты в рублях |
| `Показы` | UInt64 | Количество показов |
| `Клики` | UInt64 | Количество кликов |
| `Конверсии` | UInt64 | Количество конверсий |
| `CR, %` | Float64 | Conversion Rate, % |
| `CPA, ₽` | Float64 | Cost Per Action, ₽ |
| `CTR, %` | Float64 | Click-Through Rate, % |

## Использование

Датасет используется как основа для синтетической генерации 100К+ записей через `src/data/generator.py`. Генератор извлекает из реальных данных распределения кампаний, групп, регионов и ключевых слов, и создаёт новые строки с реалистичными метриками.

## Примечание

В файле присутствует строка-заголовок и строка с единицами измерения (пропускается через `skiprows=[1]` в `loader.py`).
